# Website of a manicure studio

## Project Overview

The project includes: fully responsive website with photo galleries, components with offers and comments.

## Tech/framework used

* react: 17.0.2
* react-router-dom: 5.3.0
* styled-components: 5.3.0
* react-helmet: 6.1.0
* react-icons: 4.2.0
* react-scroll: 1.8.4
